# Jatayuu — Marketing Site

This repository contains a Vite + React (TypeScript) marketing website for the "Jatayuu" project.

## Quick Start

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Project Structure

- `index.html` — app entry HTML
- `src/` — React source files
- `src/components/` — reusable UI and page components
- `public/` — static assets copied to the build
- `build/` — output of the production build (ignored by git)

## Notes

- This project uses Vite for fast dev experience and bundling.
- Environment variables should be placed in `.env` files (not committed).

## Contributing

Open issues or submit a PR. For quick checks, run the dev server and verify pages.
